# Instruments de musique virtuels

*Julien DE BOLLIVIER, Julien VERSTRAETE, Coline GIRARD, Baptiste HILAIRE,
Lalie HOUDRY et Baptiste LE BOULCH*

Projet d'application musicale du Master ATIAM : Conception d'instruments auto-oscillants virtuels à partie de modèles physiques, dotés de dispositifs de contrôle en temps réel. 

<ins> Organisation du repo : </ins>

- Patches MaxMSP : deux modèles de clarinette et de violon fonctionnant en temps réel, codés en Max/MSP
- Contrôle tablette graphique : les codes et interfaces pour lier l'ordinateur aux contrôleurs, Tablette gra^phique Wacom® et clavier MIDI
- Modèles physique Python : quatre modèles physique de clarinette et trois modèles de cordes frottées, fichiers Python
- Descripteurs_SVM_Cartographie : liste de descripteurs utilisés et outil de cartographie SVM avec affinage codé en Python
- Audios : des exemples audios des sons obtenus
